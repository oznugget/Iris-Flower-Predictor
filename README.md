# Iris FLower Predictor

Iris Predictor is a trained machine learning application designed to display some data from the Iris data set and to classify an unknown iris flower based on user input.

## Features
* **Interactive CLI:** Prompts the user for sepal and petal measurements.
* **Visualization:** Generates a side-by-side Matplotlib scatter plot showing exactly where your input lands compared to the rest of the dataset.

## Usage:
Install the packages in the packages file
Run the predictor.py in the command line using: python predictor.py

Input the flower measurements when prompted
